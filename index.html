TESjs

Streamlining the use of Twitch EventSub

Easily subscribe to Twitch EventSub topics. Quickly start up a webhooks endpoint and listen for Twitch EventSub notifications.

Visit TESjs:
NPM GitHub

Skip the Hard Stuff

TESjs makes it easy to get up and running by taking care of the tough stuff. Under the hood, TESjs uses Express to create a wehbooks endpoint for Twitch to send notifications to. It also takes care of the tedious signature verification so you can focus on building your awesome app!

Integrate Seamlessly

Already running an Express server in your app? TESjs has you covered! Just pass the Express object into TESjs' config and it will build a webhooks route on top of your existing server!

Install

 
npm install tesjs

Initialise a TES Instance

 
                            
const TES = require('tesjs');

// initialize TESjs
const tes = new TES({
    identity: {
        id: YOUR_CLIENT_ID,
        secret: YOUR_CLIENT_SECRET //do not ship this in plaintext!! use environment variables so this does not get exposed
    },
    listener: {
        baseURL: "https://example.com"
    }
});
